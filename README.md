## 🚀 Overview

This project demonstrates a complete data science pipeline built in **R**:

- 🔍 **Scrapes dynamic HTML and API data** using `rvest` and `httr`
- 🧼 **Cleans and wrangles** scraped content with `dplyr`, `tidyr`
- 📈 **Visualizes** insights using interactive **Tableau dashboards**
- 🔄 Reduces manual effort by 80% and enhances decision-making

---

## 🧰 Tech Stack

| Category        | Tools Used                                      |

| Visualization   | Tableau (LOD Expressions, Parameter Filters)     |
